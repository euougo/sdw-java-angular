# sdw-java-angular

## Santander Dev Week - Java + Angular

### Projeto: Tela de cotações - Santander

### Aula 1: Configurando Ambiente.

### Back-end: 
1. Instalar JDK (Oracle JDK 8).
2. Instalar intelliJ.
3. Configurar intelliJ.

### Front-end:
1. Node.js (LTS 14.16.1).
    ```
    node -v
    npm -v 
    ```
2. Visual Studio Code.
    - Desabilitar mini-map.
    - Mudar o terminal de cmd to git bash.
    - Extensões:
        - Angular Language Service.
        - TSLint.
    - Extensões opcionais:
        - EditorConfig for VS Code.
        - Atom One Dark Theme.
        - Material Icon Theme.
        - Auto Import.
        - Rainbow Brackets.
        - px to rem.

3. Angular CLI.
    ```
    npm install -g @angular/cli
    ng -version
    ng --version
    ng new hello-angular
    cd hello-angular
    ng serve --open
    ```
